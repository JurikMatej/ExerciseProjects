# Microposts

## Usage

## Webpack
### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

### Build
Compile and build

```sh
$ npm run build
```

## More Info
### Json Server

Use:
```sh
$ npm run json-server
```
To start up the "rest api"
